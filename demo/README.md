# Demo Video

This folder contains the Framecraft config for a short Agent Interview Arena explainer.

Render from the repo root:

```bash
uv run --project /Users/srinivasvaddi/Projects/framecraft python /Users/srinivasvaddi/Projects/framecraft/framecraft.py render demo/arena-demo.scenes.json --auto-duration
```

Output:

- `public/agent-interview-arena-demo.mp4`

The video explains the interview model: locked task, agent room, operator prompts, PR submission, deterministic evaluation, leaderboard.
